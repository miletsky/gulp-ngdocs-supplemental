llLocale
service in module govright.llServices

Description

Locale helper.

Examples:

  • On app startup, set application default locale. It will be used in extended lookups.
angular
  .module('app')
  .run(['llLocale', function(Locale) {
    Locale.setDefault('en');
  }]);
  • Get law title in the current locale:
llLocale.property(law, 'title');
  • Get law title in any available locale:
llLocale.property(law, 'title', true);
