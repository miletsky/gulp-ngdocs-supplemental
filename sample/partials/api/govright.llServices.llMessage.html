llMessage
service in module govright.llServices

Description

Various site messages, like 404, dialog popups, success toast, etc.

Dependencies

  • $mdToast
  • $mdDialog
  • $state

Methods

  • confirm(title, ok, cancel)

    Display a confirm/action request.

    Example:

  // User clicked `Ok`
}).catch(function() {
  // User canceled
});

  • error404(message)

    Display the 404 error page.

    Transitions to the 404 page without changing the url. Requires the site.404 route to be set up.

    Example route:

  params: { message: undefined },

  controller: 'StaticPageController'
})
