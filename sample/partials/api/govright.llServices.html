The govright.llServices module provides services that encapsulate common techniques of interacting with the GovRight Corpus API

Module dependencies

  • govright.corpusServices
  • ui.router
  • gettext
  • ngMaterial

Example

 <!doctype html>
 <html ng-app="myApp">
   <head>
     <script src="js/angular.js"></script>
     <!-- Include the ll services script -->
     <script src="dist/govright-ll-services.js"></script>
     <script>
       // ...and add 'govright.llServices' as a dependency
       var myApp = angular.module('myApp', ['govright.llServices']);
     </script>
   </head>
   <body></body>
 </html>
